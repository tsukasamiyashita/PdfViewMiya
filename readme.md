# PdfEditMiya

起動と描画の高速化に特化し、スクロール遅延ロード（Lazy Loading）を採用したデスクトップPDFビューアです。1ファイルで動作しつつ、一般的なPDF閲覧アプリと同等の機能を備えています。

**バージョン**: v1.0.0

## 実装されている機能とアップデート
- **タイトル変更**: アプリケーションの名前を「PdfEditMiya」に変更しました。
- **メニューバーとヘルプ機能**: 画面上部のメニューバーの「ヘルプ」から「Readme」の確認や、「バージョン情報 (v1.0.0)」のダイアログを開くことができます。
- **マウスホイールによるページ切り替え**: 単一ページ表示モード時に、ページ最上部・最下部に達している状態でマウスホイールを回すと、直感的に「前のページ」「次のページ」へ切り替わります。
- **A3・A4混在ドキュメントの自動サイズ調整**: 単一ページ表示モード時、ページをめくるごとに「現在のページの物理サイズ（A3やA4）」を検知し、常に画面ぴったりに収まるようズーム倍率を自動再計算します。
- **起動時の最大化表示**: アプリケーションを起動すると自動的にウィンドウが最大化され、広い画面で快適にPDFを閲覧できます。
- **キーボードナビゲーション**:
  - **上 / 下 矢印キー**: 1回あたりのスクロール量を大幅に増やし（約150px）、サクサクと読み進められます。
  - **左 / 右 矢印キー**: それぞれ「前のページ」「次のページ」へジャンプします。
- **単一ページ表示モード**: ツールバーの「📄 単一ページ表示」ボタンから、縦スクロールによる連続表示モードと、1ページずつ独立して表示するモードをシームレスに切り替えられます。
- **高度なスクロール操作**: 
  - **Ctrl + マウスホイール**: 直感的なズームイン・ズームアウト。
  - **Shift + マウスホイール**: 水平方向（横）へのスクロール。
  - **Space / Shift + Space**: 1画面分の一括スクロール（ページダウン/ページアップ）。
- **ドラッグで画面移動（パン機能）**: マウスの左クリックを押しながらドラッグすることで、直感的に画面を全方向にスクロール・移動できます（手のひらツール）。
- **目次表示**: ツールバーの「📑 目次」ボタンからサイドバーの目次をトグル表示し、クリックで該当ページへ瞬時にジャンプします。
- **回転（Rotation）**: 90度単位での右回転・左回転機能。
- **印刷**: OSネイティブの印刷ダイアログからの高解像度印刷。
