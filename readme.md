# PDF Viewer App

起動と描画の高速化に特化し、スクロール遅延ロード（Lazy Loading）を採用したデスクトップPDFビューアです。1ファイルで動作しつつ、一般的なPDF閲覧アプリと同等の機能を備えています。

## 実装されている機能とアップデート
- **ドラッグで画面移動（パン機能）**: マウスの左クリックを押しながらドラッグすることで、直感的にPDFを全方向にスクロール・移動できるようになりました（手のひらツール）。
- **目次の初期非表示**: ツールバーの「📑 目次」ボタンからいつでも呼び出せますが、初期状態では表示領域を広く確保するために非表示に設定しました。
- **高速描画・白紙化防止**: 現在画面に表示されている領域だけをレンダリングし、安全なバイナリ変換によりメモリ破棄を防ぎます。
- **ページナビゲーション**: 現在のページ番号のリアルタイム表示と、数値入力によるページ移動。
- **幅に合わせる（Fit to Width）**: ウィンドウの幅に合わせて自動的にズーム倍率を最適化します。
- **回転（Rotation）**: 90度単位での右回転・左回転機能。
- **拡大・縮小**: 任意の倍率でのズーム操作。
- **印刷**: OSネイティブの印刷ダイアログからの高解像度印刷。
